Detailed Description

The TPCReadoutInfo contains information about the readout structure like number of virutal channels, hardware channels, mapping, an array of TPCChannelInfos etc.

The mapping is initialised with the trivial (1:1) mapping

void TPCReadoutInfo::ReadChannelInfoFromFile ( const char *  filename)

Read the channel information from a file.

File Syntax: There might be a header with additional information. The relevant information starts with a line containing "dead channels:" (without quotation marks). All following lines contain one integer with the (virtual) channel number. The second block starts with the tag "noisy channels:"

Example:

This is header in formation which is ignored!

dead channels:
0
1
62
63

noisy channels:
17
23
